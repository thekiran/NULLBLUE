# NULLBLUE

**NULLBLUE** is a Bluetooth Security Research & Lab Framework designed for
controlled environments, academic study, and defensive security analysis.

---

## Purpose

- Bluetooth protocol research
- Lab-based security experiments
- Stability & resilience observation
- Metrics-driven analysis

> ⚠️ This project is intended **only for authorized lab environments**.
